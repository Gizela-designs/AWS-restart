# AWS re/Start Program 

This repository documents my learning journey through the **AWS re/Start Program**, covering core AWS services, concepts, and hands-on labs.  
Each folder contains notes, exercises, and summaries for the specific service category.

---

##  Current Modules

### 1. AWS Compute Services
- **Amazon EC2:** Virtual servers in the cloud  
- **AWS Lambda:** Serverless computing  
- **Elastic Beanstalk:** Managed application deployment  

### 2. AWS Storage Services
- **Amazon S3:** Object storage  
- **Amazon EBS:** Block storage for EC2  
- **Amazon EFS:** Scalable file storage  
- **AWS Backup & Glacier:** Data protection and archiving  

### 3. AWS Database & Analytics
- **Amazon RDS:** Managed relational databases  
- **Amazon DynamoDB:** NoSQL database service  
- **Amazon Redshift:** Data warehousing  
- **Amazon Athena:** Query data directly in S3
